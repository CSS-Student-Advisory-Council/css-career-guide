Curated for CSS Students

This guide shares perspectives from CSS students on preparing for internships, as well as student clubs and resources (on and off-campus) to check out!

Hosted by the Student Advisory Council and Computing Careers Club

The Student Advisory Council is a student organization focused on student curricular feedback and connecting with faculty to improve the CSS curriculum and program. In Spring 2022, the Computing Careers Club was formed to encompass SAC's student-facing projects, including managing this guide.

Open to Contributions

We are currently accepting contributions on our GitHub . Check out or create an issue about topics to expand on in the guide!
